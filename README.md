# Todo App

A todo application built with HTML/CSS and Javascript

## About the Project

This project is a browser-based todo application built with JavaScript and bundled with Webpack. It allows users to organize tasks into folders, assign priorities and due dates, edit existing tasks, mark tasks as completed, and delete tasks or folders.

The application stores all data using the browser's localStorage API, allowing tasks and folders to persist between sessions without requiring a backend or database.

This project was built to strengthen my understanding of object-oriented JavaScript, DOM manipulation, modular application structure, event handling, and client-side data persistence.
## Built With

- JavaScript
- HTML
- CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/todo-app.git
cd todo-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

Build for production:

```bash
npm run build
```
